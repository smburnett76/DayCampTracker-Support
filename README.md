# Camp Track — Support

Public support and legal information for **[Camp Track](https://apps.apple.com)** on the iPhone.

Camp Track helps parents manage summer camp schedules—kids, camps, calendar, conflicts, and local reminders—all stored on your device.

## About the app

- **Platform:** iOS  
- **Bundle ID:** `com.camptrack.app`  
- **Data:** Your family’s schedule stays on your iPhone; we do not operate servers that store your camp data.

Source code is maintained in a separate private repository.

## Enable GitHub Pages (one-time)

1. Push this repo to GitHub as **`CampTrack-Support`** (public recommended so Apple can reach your policy URL).
2. On GitHub: **Settings → Pages**.
3. **Build from branch:** `main` (or `master`), folder **`/ (root)`**.
4. Save. After a minute or two, your site is live at  
   `https://YOUR_GITHUB_USERNAME.github.io/CampTrack-Support/`

Use the privacy policy link above in App Store Connect → **App Information → Privacy Policy URL**.  
Use this README page (root URL) for **Support URL** if you do not have a separate website.
